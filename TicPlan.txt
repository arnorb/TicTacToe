Tic Tac Toe: The Battle plan


Hvað þarf tic tac toe?

Leikjaborð:
	- 3x3 að stærð
	- Hugsanlegt að útfæra í command line
	- Hugsanlegt að útfæra með griddi af 9 tökkum.
	- Athuga hluti eins og java.awt.*, java.awt.event.* og java.swing.* sem hafa innbyggð grafísk element?
	- Athuga kannski með html útfærslu svo hægt sé að gera selenium test? Nema við viljum ekki fá hærri einkunn en 6...

Interaction:
	Ef CL leið er valin
	- Leikborð ætti helst að prentast, þá helst merkt. Dæmi:
	 
	 | 1 | 2 | 3 |
	_|___|___|___|_
	 |   |   |   |
	A| X |   | O |
	_|___|___|___|_
	 |   |   |   |
	B| X | O |   |
	_|___|___|___|_
	 |   |   |   |
	C|   | X |   |
	_|___|___|___|_
	 |   |   |   |

	- Þá getur leikmaður slegið inn reit á CL, td "A3", og þá birtist hans tákn þar.
	- Eftir hvern leik prentast borðið út.

	Ef grafísk leið er valin
	- Þegar leikmaður smellir á reit birtist hans tákn þar.

	Sameiginlegir eiginleikar
	- Það þarf að merkja reitinn með x eða o, eftir því hvort hann gerir fyrstur (x) eður ei (o). Heppileg leið er bara að nota %2.
	- Eftir reitur hefur verið valin má ekki velja hann aftur.


Útreikningar:
	- Reikna þarf út sigurvegann þegar hann hefur náð þremur reitum í röð.
	- Jafntefli ef allir reitir fyllast án 3 reita í röð
	- Ekki er mögulegt að neinn vinni fyrr en í 5. leik, þeas þegar einn leikmaður er með 3 reiti
	- Það þarf að skanna línurnar eftir vinningshafa eftir hvern leik uppúr 5. leik

	Tillaga á útfærslu á leikborði og útreikning á vinningsstöðu:
		Við gætum einvaldlega notað tvívítt fylki fyrir reitina. Þá erum við með:
	      0 1 2
		0 ▓ ▓ ▓
		1 ▓ ▓ ▓
		2 ▓ ▓ ▓

		Eftir 5. leik þá athugum við hvort sigurleikur hafi verið spilaður.
		T.d ef leikmaður hakar við reit 0,0. Þá þurfum við að athuga lóðrétta og lárétta línu út frá 0,0, en þar sem skálína er möguleg athugum við hana líka. Þá þurfum við ekki að skanna allt borðið, bara línurnar sem tengjast síðasta leik. Ef þrír reitir merktir leikmanni finnast, SIGUR!!!
		Við getum þá td táknað x með 1 og o með 2 í fylkinu, en auðan reit líklegast með 0.

		Hins vegar gæti verið að aðrar leiðir séu í boði ef við notum einfaldlega grafísku leiðina varðandi fylkið og svona. Er ekki búinn að skoða það almennilega.
